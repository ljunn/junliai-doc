# 费用与订阅指南

这一页主要回答 3 个核心问题：
1. 余额和订阅有什么区别？
2. 套餐应该怎么买、怎么选？
3. API 使用有哪些限制？

---

## 1. 两种使用方式

::: info 提示
根据你的使用频率，可以选择不同的计费模式。
:::

*   **余额充值**：按量付费，适合低频或不定期使用的场景。
*   **包月订阅**：按月付费。根据套餐档位不同，每天享受不同的额度上限，适合高频开发者。

---

## 2. 获取方式

你可以通过以下渠道获取服务额度：

**在线充值**：直接在管理后台进行余额充值。

::: tip 兑换说明
如果你持有购买的兑换码，请登录后台进入“兑换中心”进行激活即可。
:::

## 3. 并发与限制规则

为了保障链路稳定，系统执行以下并发规则：

*   **并发叠加**：如果你购买了多个不同类型的套餐，其并发数可以叠加。
*   **套餐限制**：单一套餐内的调用将严格按照该套餐定义的并发频率进行限制。

::: warning 注意
`包月订阅` 和 `余额充值` 的 API Key 通常是不通用的，创建 Key 时请务必选择正确的分组。
:::
